# Code 1（STL）

**你告诉我不用 STL 解这道题？除非我是二货。**

`std::pow()`

```C++
class Solution {
public:
	int mySqrt(int x) {
		return std::pow(x, 0.5);
	}
};
```

# Code 2

`sqrt() 定义于 "math.h"`

```C++
class Solution {
public:
	int mySqrt(int x) {
		return std::sqrt(x);
	}
};
```
