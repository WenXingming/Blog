# 使用前的环境配置

```bash
# 安装 rake
gem install rake
# 查看 rake 是否安装及版本
rake --version
```

# 使用命令格式

更多请 `rake --help` 查看。一般使用：

```bash
rake post title="Hello 2015" subtitle="Hello World, Hello Blog"
# 发现有个缺陷是：md 中 front config 中的 date 只有日期，没有时间，需要补上
```
